# Models

## File Descriptions

+ PVC to Minivac: connect a T-joint of a vacbed frame to 3/16" tubing for use with a distribution block.
+ PVC to Dual Minivac: connect a T-joint of a vacbed frame to 3/16" tubing for direct use with an inverted aquarium pump.
+ Tube Comb: organizes your long doubled-up 3/16" tubing runs

## Printing

All models in this directory are designed to be printed in PETG.

The settings aren't terribly important - they all worked fine on a 10+ year old printer with a 0.6mm nozzle.
